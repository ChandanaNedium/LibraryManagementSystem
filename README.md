# LibraryManagementSystem

The Library Management System is designed to automate library operations such as managing books, issuing books, and returning books. The system has two modules: Admin and User.

//Features

Admin:

Add new books

View book records

User:

Search books

Issue books

Return books

//Technologies Used

Java

ArrayList

Scanner class

//How to Run

Compile:

javac LibraryManagementSystem.java

Run:

java LibraryManagementSystem

//Sample Output
===== LIBRARY MANAGEMENT SYSTEM =====
1 Admin Login
2 User Section
3 Exit
